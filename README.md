# МамаХаб — Социальная сеть для будущих мам

Веб-приложение для беременных: общение, музыка, трекер беременности, объявления, клипы, гороскоп и многое другое.

## Технологии

- **React 18** + **TypeScript**
- **Vite** — сборка и dev-сервер
- **Tailwind CSS** + **shadcn/ui** — стилизация и UI-компоненты
- **Framer Motion** — анимации
- **React Router** — маршрутизация
- **React Query** — управление серверным состоянием
- **Recharts** — графики
- **Radix UI** — доступные примитивы интерфейса

## Возможности

- **Лента** — публикации, лайки, комментарии
- **Друзья** — поиск, заявки, рекомендации
- **Клипы** — короткие видео
- **Группы** — тематические сообщества по триместрам
- **Музыка** — плеер с плейлистами (колыбельные, медитация, классика)
- **Сообщения** — личная переписка
- **Объявления** — доска детских товаров
- **Трекер беременности** — развитие малыша по неделям
- **Гороскоп зачатия** — знак зодиака малыша
- **Профиль** — аватар, статистика, публикации
- **Админ-панель** — темы оформления, управление пользователями, статистика
- **Кастомизация тем** — нежные, тёплые, холодные, тёмные, яркие, природные, классические

## Установка и запуск

### Требования

- **Node.js** >= 18
- **npm** >= 9

### Шаги

1. **Клонировать репозиторий:**

```bash
git clone https://github.com/micleberry556-eng/Beby.git
cd Beby
```

2. **Установить зависимости:**

```bash
npm install
```

3. **Запустить dev-сервер:**

```bash
npm run dev
```

Приложение будет доступно по адресу: [http://localhost:8080](http://localhost:8080)

### Другие команды

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск dev-сервера с HMR |
| `npm run build` | Сборка для продакшена |
| `npm run preview` | Предпросмотр продакшен-сборки |
| `npm run lint` | Проверка кода линтером |
| `npm run test` | Запуск тестов |
| `npm run test:watch` | Запуск тестов в watch-режиме |
